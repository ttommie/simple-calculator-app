# Hello-World-React

A simple calculator project created using react natives.

## Features

-   Real-time Input & Calculation Updates.
-   Live Preview
-   Mobile Accessible

## Deployment

Clone or Download

```bash
git clone https://github.com/ttommie/hello-world-react.git
```

Install all node modules

```bash
npm install
```

Run the react server

```bash
npm start
```
